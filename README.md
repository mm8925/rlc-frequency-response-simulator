# RLC Frequency Response Simulator

## Overview
This project simulates the frequency response of a series RLC circuit and visualizes it using animated 2D plots. It demonstrates how RLC circuits behave as frequency selective networks.

## Features
- Animated Bode magnitude plot
- Resonant frequency detection
- Frequency sweep visualization
- Clear graphical output

## Tech Stack
- Python
- NumPy
- Matplotlib

## Installation / Setup

1. Clone the repository:
   git clone https://github.com/mm8925/rlc-frequency-response-simulator.git

2. Go to folder:
   cd rlc-frequency-response-simulator

3. Install libraries:
   pip install numpy matplotlib

4. Run:
   python main.py

## Working
- Generates frequency range
- Calculates RLC response
- Converts to dB scale
- Animates the response
